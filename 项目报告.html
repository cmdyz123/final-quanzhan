项目简介

What is NutriSnap

NutriSnap 是一款基于 AI 的智能营养管理 Web 应用。
用户通过 拍照上传食物图片，AI 自动识别食物种类，计算营养成分，生成饮食报告。
内置 200+ 种中式食物营养数据库，覆盖米饭面条、盖饭煲仔、火锅麻辣、烧腊卤味等中国日常饮食。
从 "拍一张照片" 开始，让每一餐都吃得明明白白。

AI 能力详解

DeepSeek 大模型驱动

🍱 食物图像识别
  • 拍照上传，自动识别食物种类
  • 色彩分析 + DeepSeek 文本推理双通道
  • 兼容视觉 API (GPT-4o) 和纯文本 API
  • 200+ 中式食物数据库精准匹配
💬 AI 营养师对话
  • 个性化多轮对话，记住目标和偏好
  • 结合当日饮食数据针对性建议
  • 支持减重/增肌/维持等多场景
  • 全站浮动聊天窗口，随时提问
📋 智能饮食计划
  • 根据目标热量自动生成三餐搭配
  • 考虑已摄入量，智能计算剩余
  • 每餐含食物/分量/热量/贴士
  • 支持 API 模式 AIGC 个性化生成
📝 每日 AI 总结
  • 三餐记录完毕自动触发 AI 总结
  • 包含回顾、评估、建议、评分
  • 可查看任意历史日期的总结
  • DeepSeek 实时生成个性化内容

功能亮点

Feature Highlights

⚡ 快速文字输入

输入"猪脚饭加卤蛋"，AI 自动拆分为猪脚饭 + 卤蛋，识别、计算营养、保存一步到位。支持「加、+、和、配」等多种分隔符。

⭐ 常吃食物推荐

自动统计用户高频食物和常见搭配。点击按钮一键复用，告别重复输入。新用户数据不足时显示空状态引导。

🤖 全站浮动答疑

右下角 AI 聊天窗口，任何页面随时提问。预设快捷问题，DeepSeek 实时回复。打字动画、移动端适配。

📅 历史日期导航

仪表盘和记录页支持前后翻页，查看任意历史日期的饮食数据和 AI 总结。一键回到今天。

API 接口一览

10 REST Endpoints

方法路径功能说明
GET /api/search-food食物搜索 ?q=关键词 → 模糊匹配200+ 数据库
POST /api/quick-record快速记录 {"text": "猪脚饭加卤蛋"} → 拆分+保存
POST /api/manual-record手动记录 {"foods": [...], "meal_type"} → 逐项保存
POST /api/analyze-food图片分析 multipart image → AI识别+营养计算
POST /api/chatAI 对话 {"message": "…"} → 营养师回复
GET /api/daily-summary每日总结 ?date=2026-06-16 → AI 生成总结
GET /api/frequent-foods常吃统计 ?n=50&top_n=6 → 高频食物
GET /api/daily-stats每日统计 → 今日营养汇总+目标对比
POST /api/meal/<id>/append追加食物 {"text": "卤蛋"} → 合并到已有餐食
DELETE /api/meal/<id>删除餐食 → 删除整条餐食记录
